Wire schemas, hop by hop

The data shape at every arrow. Grammars marked PRIOR ART are BJJ-era and their generalized successors may amend them — recorded here so the diagram never silently upgrades an ancestor into law.

H1 — operator → Stage 0

Raw source as a direct chat attachment (zip if transformed). Never via project knowledge — the platform silently replaces PDFs with lossy text.

H2 — registration line (sources.md)

- <slug> | <type> | <title / author / year> | loc-grammar: <grammar>

Sample: - sailing-for-dummies | book | Sailing For Dummies, 2nd Edition / J.J. Isler & Peter Isler / 2006 | loc-grammar: page:N (parent chapter-N)

H3 — ingest → substrate → fidelity gate

Plain-text substrate + measurements (lines / words / book pages). Gate verdict is per content class, ruled by the operator (sailing precedent: PASS prose, FAIL tabular + figure; accepted as-is under the split verdict).

H4 — discovery ↔ operator

Button decisions with the re-ask rule; every convention line lands with one of the five provenance tags; unanswerable questions leave as escalations/ files, never silent defaults.

H5 — discovery output family → consumers

No artifact is consumed by everyone: master doc → humans + sessions (almost never workers) · harvest brief (≤25 lines) → worker packets verbatim · content-mode map (mode ∈ prose | tabular | dialogic | code_listing | mixed, no improvised values) → packet builder · loc anchors → merge + citation resolution · registry queue → registry_dossier · kit nominations → workshop (kit assembly) · config fragment → schema/config sessions ONLY (never a live-config shortcut; the v2-proposed pile was fully triaged into schema v2 on 07-13b — future discovery proposals are fenced as v3-proposed) · arbitration seed (v2.5) → the harvest arbitration layer's boot state only (judge-visible under DAG, integrator-visible under fold; tiebreak rung 2; NEVER the brief, NEVER a worker packet) · chunk plan CSV (v2.5) → packet builder + driver (defines units; any overlap policy is topology-owned and consumes them).

H6 — substrate → spans

Slicing owned by the driver. span_size / span_overlap are RETIRED as constants (v1.2 item 8; schema disposition executed 07-13b) — the per-source chunk block (schema v2 §3.2.1: ruled boundary, measured fallback bound, overrides, plan CSV) replaces them; kit spans may still use provisional sizes stamped fixture-not-policy. Span-sizing ruling (RATIFIED 07-11b): parameterize — size stays a config dial and the worker carries boundary doctrine instead of a baked-in size. Overlap floor (correctness precondition, [model-knowledge, unverified]): overlap must be at least the largest single-entity footprint the source shows, or some entity is seen whole nowhere and no downstream step can recover it. Overlap is a correctness floor, not only an efficiency dial.

H7 — packet builder → workers

Packet = cached generic prompt ⊕ per-source brief verbatim ⊕ mode block ⊕ span text. custom_id = stage:unit:attempt.

H9 — merge → glossary + residue

[07-12c: SUSPENDED v1 shape. Under the DAG candidate this hop becomes extract → prove script (three-citations law: same ink / source said so / discovery ruled so — never similarity) → born-complete case files behind a phase barrier → judge tier. Under fold it dissolves into the integrator. `residue_heuristics` is dissolved either way. Autonomy (v1.2 item 3): the residue/judge actor's escalation lane is ELIMINATED — flagged rulings live in the arbitration file; the human surface is parks + halt only.] Merge is an idempotent sweep-line interval join keyed on (term-key, overlapping loc interval): it unions evidence, prefers defined over ref, and prefers a whole sighting over a boundary-cut one — which is what makes slice-edge cuts self-heal. It resolves only the mechanical leftovers (boundary cuts, exact duplicates). Everything it cannot prove — one key with clashing sense/type, one name at disjoint locations with different senses (fan-out / sense drift) — escalates untouched to harvest_residue, one sequential judgment pass; pre-resolving any of it inside merge would itself be an identity judgment. Output: per-source glossary, frozen after harvest — what the SOURCE calls things. The interface at this hop is pinned by the downstream contract (RATIFIED 07-11b), which lives in the tail of harvest_map_v1 — and, from the receiving side, by the intake grammar (harvest_residue_v1, shipped 07-11c): one RESIDUE ITEM block per unproven cluster (heuristic flag + verbatim sightings + optional pre-grepped slices), the flag a suspicion never evidence. The residue pass rules union | split | distinct | escalate under the evidence law (every verdict traceable to the packet; un-escalated coin flips and world-knowledge verdicts are the critical failure class), emits sentinel blocks, and a dumb applier writes the glossary (single writer per artifact). The future merge script derives against both ends of this hop.

H10 — glossary + registry queue → dossiers → verdicts

Dossier = evidence both ways + a lean + NO verdict (map may not judge identity). Verdict pass writes exactly one registry line per concept — PRIOR ART grammar: - <canonical> | <type> | <status> | parent | aliases | defined_at — appends reasoning to registry_rulings.md, escalates only coin flips. Single-namespace property: exactly one process mutates the registry.

H11 — frozen snapshot → node builder

Inputs: registry snapshot, generalized style contract, per-type template slots (from discovery), grep'd source slices (grep-don't-load). Batch size ~8 verification units.

H12 — node batch

PRIOR ART format (node_builder_prompt_v5): ===== NODE: <canonical> ===== + YAML frontmatter (canonical, title, type, status, sources, requires, aliases, defined_at, cards) + fixed section order per node type + one REGISTRY DELTAS block per batch, applied by the operator, never by the session.

H13 — assembly

split_nodes.sh → vault files; linter → ledger + worklists (+ forward_declined.txt whitelist); card packager → .apkg from cards frontmatter. Generated state is never hand-edited.

H14 — runtime failure loop

Fail lint → retry once with the lint report prepended → fail again → PARK (terminal until a human looks).
