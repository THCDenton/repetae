# Harvest brief — synth

What counts:
- manoeuvre: a named action the text teaches as a procedure
- part: a named physical component of the vessel

Naming weather:
- aliases appear as parenthetical glosses on first use
- headings use title case; body uses sentence case

Noise that is content:
- figure captions carry part names not present in body text

Content that looks like noise:
- margin notes are authored content, not scanning artefact

Skip on sight:
- front matter, index, acknowledgements

Local citation form:
- page.line, e.g. 42.17
- tack will fan out [discovery-forecast]
