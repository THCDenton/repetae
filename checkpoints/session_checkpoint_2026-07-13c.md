## Ground truth learned this session

- **The architecture was chat-ready by construction.** Paste-ready
  packets, stateless single-packet actors, sentinel/fence-bounded output,
  custom_id inside the packet — the manual lane cost one spec section and
  a scorer patch, not a redesign. Interfaces-first keeps paying.
- **Enum changes propagate into fixture keys silently.** The autonomy
  ruling (07-12c) invalidated the residue kit's escalate fixtures and
  broke its coverage guarantee, and nothing flagged it for two sessions —
  no document held the interaction. Lesson: when an enum changes, grep
  every kit/fixture/spec that pledges coverage over it, same session.
- **The dry self-test earns its keep.** It caught a spec/code
  disagreement (annex scoring) before anything shipped — the second time
  this project a mechanical rule caught its own author (rev counter,
  07-13b). Cheap self-tests on instruments, always.
- **"Deferred" beats a proxy.** Recording a measurement as null-with-debt
  (token deferral) is the honest-TBD posture extended from config values
  to gate metrics. An invented number in a verdict is worse than a named
  gap with a re-open trigger.
- **Ambiguous button-presses on open sub-questions get ruled loudly.**
  "Sounds good" arrived while point 4 (proxy vs deferral) was explicitly
  open; resolution = rule it in-session with the lean stated and a
  numbered docket item, never assume the button covered it. Adopt as
  practice.
