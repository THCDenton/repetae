## Ground truth learned this session

- **The docket pattern gained a middle gear: the arguments pass.** Docket →
  operator asks "what would you do and why" → per-item recommendation with
  strongest counter + confidence label → wholesale ruling. It let the
  operator ratify 13 items in one word while knowing exactly which three
  were thin (reservation-as-enum, scalar-or-map, visual_policy). Adopt as
  the offered default for large dockets: volunteer confidence labels.
- **A re-issue is the durable fix for repeat filing misses.** W10 recurred —
  the same patch missed filing twice. Patch specs survive (that lesson
  held), but the structural fix is folding patch content into a full
  re-issue so there is no separate patched-copy step to lose.
- **Version-suffixed filenames create retirement debt; rev counters don't.**
  Evidence this session: primer v2 lingered across two flags before dying;
  my own false memory of v2_4 still sitting unretired. A rev counter with a
  mechanically checkable derivation also catches its own defects — the rev
  7 → rev 10 miscount was caught by applying the counter's own stated rule.
- **Sentinel markers are an interruption liability.** The overnight death
  caught a temporary `XCHANGELOG_TAILX` marker mid-flight in the lexicon
  working copy; verify-don't-assume found it on resume. Rule adopted for
  own practice: never end a tool call with a sentinel resident in a working
  copy — complete marker-based edits within a single call.
